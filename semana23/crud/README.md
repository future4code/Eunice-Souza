express, uuid, typeorm

R - READ    - GET    - LEITURA
* Pega as informações do usuario já criado

U - UPDATE  - PUT    - ALTERAR
* Alterar informações do usuário

D - DELETE  - DELETE - DELETAR
* Deletar um usuário
